Python Inheritance Overview

In Python, inheritance is a way to establish a relationship between classes, where a class (child or subclass) inherits attributes and methods from another class (parent or superclass). This allows for code reuse and is a key feature of object-oriented programming (OOP). Python supports several types of inheritance, each with different use cases. Hereâ€™s an overview:

1. Single Inheritance

Single inheritance occurs when a subclass inherits from only one parent class.

class Animal:
    def speak(self):
        print("Animal makes a sound")

class Dog(Animal):
    def bark(self):
        print("Dog barks")

# Creating an instance of Dog
dog = Dog()
dog.speak()  # Inherited method from Animal
dog.bark()   # Method from Dog

In this case, Dog is inheriting from the Animal class.

2. Multiple Inheritance

Multiple inheritance allows a subclass to inherit from more than one parent class.

class Animal:
    def speak(self):
        print("Animal makes a sound")

class Swimmer:
    def swim(self):
        print("Swimming")

class Fish(Animal, Swimmer):
    pass

# Creating an instance of Fish
fish = Fish()
fish.speak()  # Inherited from Animal
fish.swim()   # Inherited from Swimmer

In this example, Fish inherits from both Animal and Swimmer.

3. Multilevel Inheritance

Multilevel inheritance occurs when a subclass is derived from another subclass, creating a "chain" of inheritance.

class Animal:
    def speak(self):
        print("Animal makes a sound")

class Dog(Animal):
    def bark(self):
        print("Dog barks")

class Puppy(Dog):
    def whine(self):
        print("Puppy whines")

# Creating an instance of Puppy
puppy = Puppy()
puppy.speak()  # Inherited from Animal
puppy.bark()   # Inherited from Dog
puppy.whine()  # Method from Puppy

Here, Puppy inherits from Dog, which in turn inherits from Animal.

4. Hierarchical Inheritance

Hierarchical inheritance occurs when multiple subclasses inherit from a single parent class.

class Animal:
    def speak(self):
        print("Animal makes a sound")

class Dog(Animal):
    def bark(self):
        print("Dog barks")

class Cat(Animal):
    def meow(self):
        print("Cat meows")

# Creating instances of Dog and Cat
dog = Dog()
cat = Cat()

dog.speak()  # Inherited from Animal
dog.bark()   # Method from Dog
cat.speak()  # Inherited from Animal
cat.meow()   # Method from Cat

In this case, both Dog and Cat inherit from the same parent class, Animal.

5. Hybrid Inheritance

Hybrid inheritance is a combination of two or more types of inheritance.

class Animal:
    def speak(self):
        print("Animal makes a sound")

class Swimmer:
    def swim(self):
        print("Swimming")

class Dog(Animal):
    def bark(self):
        print("Dog barks")

class Fish(Dog, Swimmer):  # Hybrid inheritance
    pass

# Creating an instance of Fish
fish = Fish()
fish.speak()  # Inherited from Animal
fish.bark()   # Inherited from Dog
fish.swim()   # Inherited from Swimmer

Here, Fish inherits from both Dog and Swimmer.

6. Interface Inheritance (Abstract Base Classes)

Python supports a concept similar to interfaces through abstract base classes (ABCs), which define methods that must be implemented by subclasses.

from abc import ABC, abstractmethod

class Animal(ABC):
    @abstractmethod
    def speak(self):
        pass

class Dog(Animal):
    def speak(self):
        print("Dog barks")

# Creating an instance of Dog
dog = Dog()
dog.speak()  # Output: Dog barks

Here, Animal is an abstract class, and any subclass (like Dog) must implement the speak() method.

Summary of Types of Inheritance

  1. Single Inheritance: A subclass inherits from one parent class.
  2. Multiple Inheritance: A subclass inherits from more than one parent class.
  3. Multilevel Inheritance: A subclass inherits from a subclass (creating a chain).
  4. Hierarchical Inheritance: Multiple subclasses inherit from a single parent class.
  5. Hybrid Inheritance: A combination of two or more types of inheritance.
  6. Interface Inheritance (Abstract Base Classes): Defining an interface using abstract base classes, ensuring subclasses implement certain methods.

Each type of inheritance serves different use cases, depending on the structure and relationships between your classes.
